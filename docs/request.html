class Request¶

An incoming HTTP request to receive data on.

method: str¶

the HTTP method for this request

url: table¶

The parse url of this request

http_version: str¶

The http version from the request preamble

headers: Headers¶

The HTTP headers for this request

body: str¶

The contents of the request

err: str or nil¶

The _last_ error from the handler or middleware

handled: boolean or nil¶

true when the request has been handled
